Adaptive CAPTCHA Service

Behavior-based human verification

This service provides an embeddable adaptive CAPTCHA widget for external websites. Website owners connect the script to their pages, and the system collects behavioral interaction data in the background. Risk evaluation is triggered only before protected actions such as login, registration, purchase, or form submission.

1. Invisible collection

The widget records mouse movement, clicks, scrolling, keyboard activity, focus changes, and page interaction events.

2. Risk scoring

Before a protected action, the collected session is sent to the backend and evaluated by the machine learning model.

3. Adaptive challenge

Low-risk users continue without interruption, while suspicious sessions receive medium or hard CAPTCHA verification.
